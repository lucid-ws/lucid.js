    public ClientStatus: Object source 

    import {ClientStatus} from 'lucid/server/constants/Constants.js'

    Describes the state of a Client.

    Properties:

    NameTypeAttributeDescription
    ACTIVE Number

    Shows that the Client is active - it is connected.
    INACTIVE Number

    Shows that the Client is inactive - it is disconnected, but has the chance to renew its session. Clients may be inactive when their connection has dropped.
    REMOVED Number

    Shows that the Client is disconnected and they are not permitted to return to their session.
    UNAUTHORIZED Number

    Shows that the Client is unauthorized. Only applies to UnknownClient.

    import {ServerStatus} from 'lucid/server/constants/Constants.js'

    Describes the state of the Server.

    Properties:

    NameTypeAttributeDescription
    AVAILABLE Number

    Shows that the server is ready/available.
    UNAVAILABLE Number

    Shows that the server is unavailable and is not attempting to start/restart.
    RESTARTING Number

    Shows that the server is unavailable but is trying to restart.
    STARTING Number

    Shows that the server is starting up for the first time.
